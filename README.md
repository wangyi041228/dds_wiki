## 主线0/教程
* Esc菜单可跳过教程，退出游戏或被杀也行。这样少三个芯片，角色名随机为`Me`或`You`。
* 报道后，进电梯前出门可看见名为`munkey`的NPC，每踩一次红色区域就多一个。
* 会用近战武器后，打破附近的玻璃可捡到两个芯片。另一个在后门后。
